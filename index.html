适用场景

AI 训练数据准备：将企业文档、技术手册转换为结构化格式，用于 AI 模型训练。

知识库构建：将 PDF 报告、Word 文档转换为 Markdown，方便导入知识库系统。

数据分析：将 Excel 表格转换为 JSON 格式，便于程序处理和 API 对接。

内容迁移：将演示文稿转换为文本格式，方便内容复用和编辑。

转换说明

Word 文档：保留标题层级、段落结构、列表格式，转换为 Markdown 时保留基本样式。

PDF 文档：自动检测文本型或扫描版 PDF。扫描版 PDF 会启用 OCR 识别（支持中英文），提取图片中的文字。

Excel 表格：转换为 Markdown 表格或 JSON 数组，保留数据和表头。

PPT 演示文稿：提取每页幻灯片的文本内容，按页面组织输出。
